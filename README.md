# business-website

Business website for Project P, based on [PaperMod](https://github.com/adityatelange/hugo-PaperMod)

Deployed at [business.tranquiltina.com](https://business.tranquiltina.com)

## Update

```bash
cd themes/PaperMod
git pull
```

## Deployment

On the server, run:

```bash
cd /business-website
git pull
hugo server
```
